Requête et transformation des données

Matériels pour les exercices

Montrer le path vers les données pour les exercices

inclus:

  • Gabarit Power PI
  • Données source Excel
  • dimDate

Contenu:

Cette formation comprend des explications sur les concepts de base de Power BI et de préparation des modèles de données.

1 - Éditeur de requête

La première chose à faire pour créer des rapports est de charger des données.

Bien que l'option la plus évidente soit de sélectionner "1- Obtenir des données", ce n'est pas recommandé.

Il est mieux d'utiliser l'option 2 -Transformer les données.

L'éditeur de requête est un outil à l'intérieur de Power BI qui :

  • Permet de transformer vos données de la forme brute à une forme optimisée pour Power BI ;
  • Est l'endroit où vous passerez 90 % de votre temps.

Comment charger des données​

  1. Sélectionner Nouvelle Source

  2. Sélectionner Excel

  3. Naviguer jusqu'au fichier Excel à charger

  4. Sélectionner le fichier à charger

  1. L'éditeur présente le navigateur avec le contenu du fichier Excel disponible pour charger dans Power BI.
  2. Différents icones sont présentés :
    • Table nommée : Une table spécifique dans Excel avec un nom
    • Onglet ou page Excel: Une page Excel

On recommande de toujours utiliser des tables nommées.

La raison est simple, cela simplifie le travail de la requête pour retrouver les données dans Excel et limite les erreurs de chargement.

  1. Sélectionner les données à charger

  1. Cela transfert les données du fichier Excel dans le fichier Power BI

Les données sont maintenant disponibles pour être transformées et améliorées

  1. L'onglet "Transformer" donne accès à plusieurs options et un clique droit sur n'importe quelle colonne offre encore plus de possibilité de transformation

Comment transformer des données Excel en tableau nommé​

  1. Sélectionnez une cellule parmis les données que vous voulez transformer en tableau

  2. Sélectionnez l'onglet "insertion"

  3. Cliquer sur l'option tableau

  4. Cliquer sur OK pour transformer la plage de données en tableau nommé

  1. Les données sont transformées en table et le visuel change pour le style de tableau actif

  2. L'onglet création est maintenant disponible et offre plusieurs options pour personnaliser le tableau

  3. Il est maintenant possible de nommer le tableau.

    C'est ce nom qui sera affiché lors du chargement dans Power BI.

    Un avantage important des tableaux est que lors du chargement, Power BI ignorera les valeurs dans les colonnes en dehors de la zone du tableau.
