Profilage-données

Profilage des données

Chaque fois que nous nous connectons à une source de données à l'aide du mode Importation, on choisit le mode transformer les données au lieu de charger les données afin que nous puissions exploiter des capacités de profilage des données pour n'importe quelle colonne de la source.

Les outils de profilage des données offrent de nouvelles méthodes intuitives pour nettoyer, transformer et comprendre les données dans l’éditeur de Power Query. Ils comprennent :

  • Qualité de la colonne
  • Distribution des colonnes
  • Profil de colonne

Pour activer les outils de profilage des données, accédez à l’onglet affichage sur le ruban. Activez les options souhaitées dans le groupe aperçu des données , comme illustré dans l’image suivante.

Une fois que vous avez activé les options, vous verrez un résultat semblable à l’image suivante dans l’éditeur de Power Query.

Par défaut, Power Query effectue ce profilage des données sur les 1 000 premières lignes de vos données. Pour qu’elle fonctionne sur l’ensemble du jeu de données, dans l’angle inférieur gauche de la fenêtre de l’éditeur, vous pouvez modifier le mode de profilage des colonnes.

Qualité de la colonne​

La fonctionnalité de qualité de colonne étiquette les valeurs en lignes en cinq catégories :

  • Valide, indiquée en vert.
  • Erreur, affichée en rouge.
  • Vide, indiqué en gris foncé.
  • Inconnu, indiqué en pointillés verts. Indique qu’il y a des erreurs dans une colonne, la qualité des données restantes est inconnue.
  • Erreur inattendue, indiquée en pointillés rouges.

Ces indicateurs s’affichent directement sous le nom de la colonne dans le cadre d’un petit graphique à barres, comme illustré dans l’image suivante.

Le nombre d’enregistrements dans chaque catégorie de qualité de colonne est également affiché sous la forme d’un pourcentage.

En plaçant le curseur sur l’une des colonnes, vous voyez la distribution numérique de la qualité des valeurs dans l’ensemble de la colonne. En outre, si vous sélectionnez le bouton de sélection (...), des boutons d’action rapides s’ouvrent pour les opérations sur les valeurs.

Distribution des colonnes​

Cette fonctionnalité fournit un ensemble d’éléments visuels sous les noms des colonnes qui illustrent la fréquence et la distribution des valeurs dans chacune des colonnes. Les données de ces visualisations sont triées dans l’ordre décroissant de la valeur avec la fréquence la plus élevée.

En pointant sur les données de distribution dans n’importe quelle colonne, vous pouvez obtenir des informations sur les données globales dans la colonne (avec des valeurs de nombre et uniques). Vous pouvez également sélectionner le bouton des points de suspension et choisir dans un menu d’opérations disponibles.

Profil de colonne​

Cette fonctionnalité fournit une analyse plus approfondie des données dans une colonne. En dehors du graphique de distribution de colonnes, il contient un graphique de statistiques de colonnes. Ces informations s’affichent sous la section aperçu des données, comme illustré dans l’image suivante.

Filtrer par valeur​

Vous pouvez interagir avec le graphique de distribution de valeurs sur le côté droit et sélectionner l’une des barres en pointant sur les parties du graphique.

Cliquez avec le bouton droit pour afficher un ensemble de transformations disponibles pour cette valeur.

Copier des données​

Dans le coin supérieur droit des sections statistiques de colonne et distribution de valeurs, vous pouvez sélectionner le bouton de sélection (...) pour afficher un menu contextuel de copie . Sélectionnez-la pour copier les données affichées dans les deux sections dans le presse-papiers.

Regrouper par valeur​

Lorsque vous sélectionnez le bouton de sélection (...) dans l’angle supérieur droit du graphique de distribution de valeurs, en plus de copier , vous pouvez sélectionner Grouper par. Cette fonctionnalité regroupe les valeurs de votre graphique selon un ensemble d’options disponibles.

L’image ci-dessous montre une colonne des noms de produits qui ont été regroupés par longueur de texte. Une fois que les valeurs ont été regroupées dans le graphique, vous pouvez interagir avec des valeurs individuelles dans le graphique, comme décrit dans Filtrer par valeur.
