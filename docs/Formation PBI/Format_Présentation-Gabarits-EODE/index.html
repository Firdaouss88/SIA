Présentation des Gabarits EODE Power BI Desktop​

C'est quoi un gabarit Power BI (.pbit) ?​

Un gabarit ou modèle Power BI desktop est un fichier Power BI desktop sauvegardé avec l'extension .pbit. Il permet de définir des références visuelles et de les intégrer dans un rapport modèle pour faciliter et normaliser la conception de rapport.

L'objectif d'un gabarit est de faciliter, accélérer et normaliser la création de rapports Power BI.

Un gabarit inclut normalement :

  • Des normes ou références graphiques définies sous un format de fichier .json ;
  • Un format/taille de page prédéfini ;
  • un fond d'écran adapté au norme et couleur de l'entreprise ;
  • Des guides et aide structurant pour faciliter la mise en page.

Les extensions de Fichier Power BI Desktop​

.pbix = un rapport Power BI Desktop .pbit = un gabarit ou modèle Power BI Desktop

Lors de la première sauvegarde, le gabarit vous demandera de saisir un nom de rapport et changera l'extension de .pbit à .pbix. Vous passerez alors du gabarit à votre rapport.

Présentation du gabarit Power BI Desktop - Rapport et tableau de bord​

Le gabarit Power BI Desktop vient avec un fichier .json préchargé et configuré. Il inclut aussi un fond d'écran, la position du titre et des slicers par défaut ainsi qu'un guide d'alignement. Toutes ces options prédéfinies ont pour but de:

  • Normaliser
  • Accélérer
  • Minimiser

les formatages requis lors de la conception de rapport

Page de départ​

  1. Titre - Position du sujet de la page et du titre rapport
  2. Slicer - Option d'un slicer déjà positionné
  3. Fond écran - Intégré au fichier .json, le fond d'écran est prédéfini
  4. Marge - Un guide de référence pour aider à positionner rapidement les visualisations
  5. Sans source de données - Aucun source de données n'est définie dans le gabarit
