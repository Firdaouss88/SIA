Les différents types de filtres dans Power BI​

Il existe trois types de filtres dans Power BI. Cela signifie que vous pouvez appliquer des filtres à trois niveaux différents dans un tableau de bord ou un rapport Power BI. Les types de filtres sont les suivants :

  1. Filtres de niveau visuel Les filtres appliqués au niveau visuel sont appliqués directement sur les visualisations individuelles. Ces filtres sont appliqués à la fois sur les données et les conditions de calcul utilisées dans une visualisation.
  2. Filtres de niveau page Les filtres de niveau page concernent une page particulière d'un rapport, alors que les rapports comportent généralement plusieurs pages. Vous pouvez appliquer certaines conditions de filtrage sur une page sélectionnée dans un rapport. Chaque page d'un rapport peut se voir appliquer un ensemble différent de conditions de filtrage.
  3. Filtres au niveau du rapport Les filtres au niveau du rapport sont les filtres que vous utilisez pour appliquer une condition de filtre sur l'ensemble du rapport. Le filtre de niveau rapport sera appliqué à chaque visualisation et page d'un rapport. Ainsi, contrairement aux filtres de niveau visuel et de niveau page, les filtres de niveau rapport sont des filtres généralisés.

L’affichage d’une page de rapport​

Vous pouvez afficher les rapports sur différents appareils dont les tailles d’écran et les proportions peuvent varier. Changez le mode d’affichage d’une page de rapport pour répondre à vos besoins.

  1. Si vous visualisez un rapport sur un petit appareil et vous avez du mal à voir les titres et les légendes. Sélectionnez Affichage > Taille réelle pour augmenter la taille de la page du rapport. Utilisez les barres de défilement pour vous déplacer dans le rapport.
  2. Cette option consiste à adapter le rapport à la largeur de votre écran en sélectionnant Ajuster à la largeur. Dans la mesure où il s’agit uniquement de la largeur et non de la hauteur, vous devrez peut-être utiliser la barre de défilement verticale après.
  3. Si vous ne souhaitez pas de barres de défilement, mais que vous voulez utiliser au mieux la taille de votre écran, sélectionnez Ajuster à la page.
  4. Vous pouvez également choisir parmi quatre Couleurs à contraste élevé : Contraste élevé n°1, Contraste élevé n°2, Noir à contraste élevé et Blanc à contraste élevé. Vous pouvez utiliser cette fonctionnalité d’accessibilité pour que les personnes malvoyantes puissent mieux voir les rapports.

Exporter des rapports Power BI​

Power BI vous permet de publier votre rapport au format PDF ou dans Microsoft PowerPoint ou dans EXCEL:

  1. Quand vous exportez votre rapport vers PowerPoint, voici ce qui se produit :
    • Chaque page du rapport Power BI devient une diapositive individuelle dans PowerPoint.
    • Chaque page du rapport Power BI est exportée en tant qu’image haute résolution dans PowerPoint.
    • Vous pouvez conserver les paramètres des filtres et des segments que vous avez ajoutés au rapport.
    • Un lien est créé dans PowerPoint vers le rapport Power BI.
  2. Dans la barre de menus, si vous sélectionnez Exporter > PDF,Une fenêtre contextuelle s’affiche, dans laquelle vous pouvez sélectionner l’option Valeurs actuelles ou Valeurs par défaut. L’option Valeurs actuelles exporte le rapport dans l’état actuel, qui inclut les modifications actives apportées aux valeurs de segment et de filtre. Vous pouvez aussi sélectionner l’option Valeurs par défaut, qui permet d’exporter le rapport dans son état d’origine, sans refléter les modifications que vous avez apportées à cet état d’origine.
  3. Exporter le rapport dans un analyseur dans Excel est pratique lorsqu’il s’agit simplement d’obtenir les données sous-jacentes à un visuel présent sur un rapport Power BI.
