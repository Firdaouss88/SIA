Hyperlien Formation Power BI

[TOC]

Les liens suivants dirigent vers des formations sur les sujets liés à Power BI. Certaines de ces formations sont payantes.

Il existe plusieurs sites de formation, ceux-ci sont retenus pour leurs qualités, inclus des exercices et le matériel de pratique. D'autres formations peuvent être aussi efficace voir meilleurs. La liste sera mise à jour lors de nouvelle découverte pertinente.

Noter que les formations payantes sont souvent offertes à prix réduit. Spécialement ceux d’Entreprise DNA qui ont parfois jusqu'à 80 % de rabais.

Noter que ceci n'est plan un plan de cours, mais une simple liste vers des cours intéressants

Lien sur des sites (plusieurs cours offerts)

  • SQL BI

    Les formations de SQL BI ne sont pas des formations d'introduction à Power BI. Ils visent en générale un public qui a déjà une base avec l'outil Power BI Desktop.

  • [Entreprise DNA](https://portal.enterprisedna.co/courses/category/Power BI)

    Enterprise DNA offre des formations variée pour tous les niveaux.

Formations gratuites​

SQL BI​

Introduction à la modélisation 108 minutes de formation en vidéo

Introduction à la modélisation de données pour Power BI est un cours vidéo d'introduction sur la modélisation de données, qui est une compétence requise pour tirer le meilleur parti de Power BI, Power Pivot pour Excel et Analysis Services. La formation s'adresse aux utilisateurs qui n'ont pas de connaissances de base en modélisation de données pour les systèmes analytiques et les rapports.

L'objectif du cours est d'introduire les principaux concepts de la modélisation dimensionnelle, en utilisant des exemples pratiques et des démonstrations pour illustrer comment obtenir le résultat souhaité sans avoir à écrire des expressions DAX complexes. La création d'un modèle de données approprié simplifie le code à écrire et améliore les performances.

Le cours est composé de 100 minutes de cours magistraux . Vous pouvez regarder les vidéos à tout moment et le système suivra vos progrès. Dans le cours, vous pouvez télécharger les diapositives et les fichiers Power BI utilisés dans les démos.

Introduction au langage DAX 3 heures de formation en vidéo

Il s'agit d'un cours vidéo d'introduction au langage DAX. DAX est le langage natif des modèles Power BI, Power Pivot pour Excel et SSAS Tabular dans Microsoft SQL Server Analysis Services. La formation s'adresse aux utilisateurs de Power BI, Power Pivot pour Excel et aux développeurs Analysis Services qui souhaitent faire les premiers pas avec DAX.

L'objectif est d'enseigner le langage DAX en utilisant Power BI pour tous les exemples. Vous pouvez télécharger le fichier Power BI Desktop initial utilisé dans les démos et répéter toutes les étapes indiquées dans les vidéos. Les mêmes concepts peuvent également être utilisés sur Power Pivot et sur Analysis Services Tabular.

Le cours est composé de 150 minutes de cours magistraux. Vous pouvez regarder les vidéos à tout moment et le système suivra vos progrès. Dans le cours, vous pouvez télécharger les diapositives et le fichier Power BI initial utilisé dans les démos.

Outils externe 129 minutes de formation en vidéo

Le cours vidéo DAX Tools enseigne comment utiliser trois outils populaires pour écrire et optimiser DAX : DAX Studio, VertiPaq Analyzer et Analyze in Excel pour Power BI Desktop. Tous ces outils sont gratuits et open source.

DAX Studio vous permet d'écrire des requêtes DAX et de tester vos mesures DAX dans un environnement où vous pouvez vérifier les résultats et mesurer les performances. Vous pouvez également utiliser DAX Studio pour exporter des données dans différents formats. VertiPaq Analyzer nécessite DAX Studio pour exporter des statistiques et des métadonnées sur un modèle de données que vous pouvez étudier plus en détail à l'aide de l'outil dédié. Vous pouvez étudier la compression et obtenir de la documentation sur toutes les expressions DAX utilisées dans un modèle à l'aide de VertiPaq Analyzer.

Analyser dans Excel pour Power BI Desktop rend très pratique le test de votre modèle Power BI à l'aide d'un tableau croisé dynamique Excel sans avoir à publier le modèle sur le service Power BI. Lors du développement de nouvelles formules DAX, cet outil raccourcit le temps d'attente et augmente votre productivité.

Le cours est composé de 129 minutes de cours magistraux.

Enterprise DNA​

Guide ultime du débutant sur Power BI 3 heures de formation vidéo

Cette formation complète et gratuite a pour but de donner à tout utilisateur novice de Power BI un aperçu de tous les concepts d'introduction à l'utilisation de Power BI comme outil d'analyse. L'objectif du cours est de travailler étape par étape à travers un scénario du monde réel afin que vous puissiez comprendre comment les données peuvent commencer à partir de leur forme la plus brute et être développées en une solution analytique convaincante à l'aide de Power BI. Power BI est bien plus qu'un simple outil de visualisation, c'est un moteur d'analyse qui peut vous aider à trouver des informations précieuses pouvant orienter la prise de décision au sein des organisations. Ce cours vise à vous apprendre comment vous pouvez y parvenir vous-même en utilisant les fonctionnalités clés de la suite Power BI.

Guide Ultime du débutant DAX 2 heures de formation en vidéo

Cette formation gratuite a été créée pour vous donner une compréhension introductive de DAX, qui est le langage de formule à l'intérieur de Power BI. DAX est la véritable puissance analytique derrière Power BI. Si vous avez bien appris, vous pouvez effectuer des analyses incroyables qui, historiquement, ont été assez difficiles à réaliser avec les outils existants (comme Excel). La formation couvre une partie de la théorie derrière DAX qu'il est important de comprendre et d'apprendre très bien avant de passer à une formule plus avancée et à l'utilisation de techniques analytiques complexes.

Formations payantes​

SQL BI​

Data Modeling 18 heures de formation en vidéo & pratique

La modélisation des données est une compétence requise pour tirer le meilleur parti de Power BI, Power Pivot pour Excel et Analysis Services. Ce cours vidéo s'adresse aux utilisateurs de Power BI Desktop ou de Power Pivot pour Excel, ainsi qu'aux développeurs Analysis Services qui souhaitent apprendre à créer le modèle de données optimal pour leurs besoins de reporting.

L'objectif du cours vidéo est d'enseigner à travers des exemples de complexité croissante comment résoudre des scénarios commerciaux en adaptant le modèle de données, de sorte que le code DAX requis devienne plus facile, plus rapide et plus robuste. Toutes les démos et les exercices sont basés sur des exemples Power BI. Cependant, les mêmes concepts peuvent être appliqués à Power Pivot et Analysis Services Tabular.

Le cours vidéo est composé de plus de 10 heures de cours magistraux, plus environ 8 heures d'exercices individuels. Vous pouvez regarder les vidéos à tout moment et le système suivra vos progrès. Dans le cours vidéo, vous pouvez télécharger le matériel pour tous les exercices.

Mastering DAX 30 heures de formation en vidéo & exercices

DAX est le langage natif des modèles Power BI, Power Pivot pour Excel et SSAS Tabular dans Microsoft SQL Server Analysis Services. La formation s'adresse aux utilisateurs de Power BI, Power Pivot pour Excel et aux développeurs Analysis Services qui souhaitent apprendre et maîtriser le langage DAX.

L'objectif du cours est d'enseigner toutes les fonctionnalités de DAX, en fournissant les connaissances nécessaires pour écrire des formules pour des scénarios commerciaux courants et avancés. Le cours vidéo comprend plus de 15 heures de cours magistraux, plus 15 à 20 heures d'exercices individuels. Vous pouvez regarder les vidéos à tout moment et le système suivra vos progrès. Dans le cours, vous pouvez télécharger le matériel pour tous les exercices.

Optimizing DAX 10 heures de formations en vidéo

Ceci est la version vidéo du cours de l'atelier Optimisation du DAX. Il vous apprend à optimiser les requêtes et les mesures dans DAX.

DAX est le langage natif des modèles Power Pivot pour Excel, Power BI Desktop et SSAS Tabular dans Microsoft SQL Server Analysis Services. Ce cours vidéo s'adresse aux utilisateurs de Power BI, Power Pivot et aux développeurs Analysis Services qui connaissent déjà le langage DAX et souhaitent apprendre à optimiser les formules et les modèles DAX pour améliorer les performances.

L'objectif du cours vidéo est de vous permettre de trouver les goulots d'étranglement dans les calculs DAX à l'aide d'outils tels que Profiler et DAX Studio, en fournissant les connaissances nécessaires pour améliorer les performances en utilisant différentes formules DAX ou en modifiant le modèle de données, si nécessaire. Le cours vidéo est composé de plus de 10 heures de cours magistraux, plus environ 8 heures d'exercices individuels. Vous pouvez regarder les vidéos à tout moment et le système suivra vos progrès. Dans le cours vidéo, vous pouvez télécharger le matériel pour tous les exercices.

Les étudiants ont accès à un espace de discussion privé où ils peuvent interagir avec les instructeurs en posant des questions liées aux cours et aux exercices.

Prérequis : Les participants doivent avoir une bonne connaissance du langage DAX et une expérience préalable de l'utilisation de Power Pivot pour Excel, de Power BI Desktop ou d'Analysis Services Tabular. Vous pouvez acquérir ces connaissances en suivant le Mastering DAX Workshop , ou en ayant une expérience équivalente dans le monde réel.

Enterprise DNA​

Maîtriser les calculs DAX 11 heures de formation vidéo

Cette formation vous fera passer d'un utilisateur débutant à un utilisateur avancé de Power BI en utilisant au maximum le puissant langage DAX. Le matériel du cours couvre toute la théorie du débutant que vous devez très bien comprendre pour exploiter la puissance analytique de DAX. À travers des scénarios du monde réel, vous découvrirez comment utiliser DAX dans la pratique pour découvrir les informations que vous souhaitez trouver dans vos données. Toutes les fonctions principales de DAX et comment les utiliser lors de l'exécution des calculs et en combinaison les unes avec les autres sont couvertes. DAX est la partie la plus difficile de Power BI à apprendre et à bien mettre en pratique. Ce cours couvre tous les principaux conseils et techniques qui, s'ils sont pratiqués, permettront à tout utilisateur de Power BI de créer des rapports et des solutions Power BI convaincants.

Modèle de formule DAX 5 heures de formation vidéo

Cette formation est entièrement axée sur l'utilisation des formules DAX dans Power BI pour résoudre des problèmes analytiques simples à très avancés. En utilisant la puissance de DAX, combinée au modèle de données Power BI, vous pouvez presque sans effort écrire une formule DAX qui peut résoudre même les questions les plus complexes posées à vos données brutes. Pour le faire efficacement, vous devez d'abord comprendre les concepts et les formules de base de DAX. À partir de là, vous pouvez vous diversifier dans un certain nombre de modèles réutilisables qui peuvent être superposés sur une variété d'ensembles de données. Ce cours vous apprend comment faire exactement cela avec des exemples complets sur une quantité considérable de scénarios analytiques auxquels vous serez confrontés tous les jours dans le monde réel.

Optimizing DAX

The Optimizing DAX course will teach best practices for developing data models to support optimal performance of your Power BI reports. In addition, you will learn how to diagnose and resolve performance bottlenecks in your DAX measures using a variety of tools and techniques. This course explains in plain language how the DAX calculation engines work, so you can understand and evaluate how your DAX coding choices affect your report performance, ultimately leading to a better user experience.

Data Transformation & Modeling 4 heures de formation vidéo

Ce cours couvre en détail l'éditeur de requêtes et comment développer des modèles de données Power BI efficaces. Lorsque vous démarrez avec Power BI, vous pouvez parfois oublier à quel point ces deux domaines au sein de Power BI sont importants pour développer une solution de reporting efficace. Ce cours permet de s'assurer que vous pouvez voir l'immense valeur du travail dans ces domaines peut-être juste. Sont abordées des techniques intermédiaires à avancer qui vous permettront d'optimiser vos tableaux de données brutes, puis de les connecter à un modèle analytique fonctionnel auquel vous pourrez finalement superposer la formule DAX et obtenir les résultats corrects, qui répondront à vos questions analytiques.
