Gérer les prérequis au libre-service

Gérer les prérequis au libre-service

L'ensemble des conditions devant être fixée avant de pouvoir faire du libre-service.

Tâches​

  1. Valider le type de projet:
    • Un ajout à un projet affaires;
      • Sans libre-service en place;
      • Avec libre-service déjà en place;
      • Un projet libre-service autonome.
  • Si projet affaires
    • Faire créer un groupe AD de super-utilisateurs ;
    • Valider si les groupes AD d'application PBI sont réutilisés pour le libre-service, sinon faire créer un groupe AD d'application PBI;
  • Si Projet libre-service
    • Faire créer un groupe AD de super-utilisateurs ;
    • si requis : faire créer un groupe AD d'application PBI sinon Les rapports libre-service ne sont pas partagés avec d'autres gens que les super-utilisateurs;

Pré -requis: Un espace de travail de jeu de données doit être en place pour le sujet du libre-service.

Si c'est le premier espace de travail pour ces super-utilisateurs ou un ajout de contexte de sécurité

  1. Faire une demande au CSU pour créer le groupe AD de tâche super-utilisateur;
  • Utiliser la nomenclature pour déterminer le nom du groupe AD;
  • lien vers la norme : Nomenclature de groupe AD;
  • lien vers la pratique : Comment faire une demande de groupe AD de super utilisateur;

Ou

Si c'est un ajout pour un contexte d'affaires sur du libre-service existant

  1. Identifier le sujet du contexte d'affaires
  • Nom du jeu de données
    • Espace de travail WS-DS
    • Nom du jeu de données spécifique
  • Identifier le nom du groupe AD de super utilisateurs
    • Lien vers la doc sur les groupes AD en production
  • Identifier le nom du contexte d'affaires pour lequel créer un WD-LB
    • Voir le questionnaire sur l'analyse de l'architecture fonctionnelle

Et

Si c'est un espace de travail de libre-service ajouté à un projet affaires

  1. Assigner le groupe AD de l'application du projet affaires en lien avec le libre service sur l'application Power BI de libre -service.
    • Liens vers la pratique: Comment gérer la sécurité d'une application Power BI;
    • Dans le cas où le libre-service est utilisé pour partager des rapports avec des utilisateurs ayant déjà des rapports corporatifs, on réutilise les groupes AD des applications Power BI existantes lorsque possible.
      • Lien vers norme : Architecture fonctionnelle, contexte de sécurité
      • Lien vers norme : Architecture fonctionnelle, contexte d'affaires
      • Lien vers... à confirmer si autre document qui explique cette situation
