SERVICE POWER BI - OPTION SÉCURITÉ

La console d'administration permet de contrôler les options du service Power BI et de déterminer lesquelles sont actives et inactives pour l'organisation.

Parmi ces options, certaines donnent accès à des fonctionnalités plus sensibles. Ces options sont alors activées et leurs accès sont contrôlés par des groupes AD spécifiques.

Dans le portail d'administration, pour chacune de ces options, elle sera :

  1. Activée.
  2. Appliquée à : Groupes de sécurité spécifiques.
  3. Un ou des groupes AD seront spécifiés.

Groupes AD & options sécurisées​

Voici la liste des groupes AD donnant accès à une ou plusieurs options sécurisées du service Power BI.

La grille suivante présente le nom des groupes AD, une brève description du rôle ciblé par le groupe et la liste des options du service Power BI auxquels le groupe AD donne accès.

Groupe AD des développeurs d'interface client (Power BI) de l'équipe corporative

Groupe AD: GR-GIE-IA-PBI-OPTIONS-DEV-AFFAIRES​

Groupe AD regroupant les fonctionnalités requises pour créer des rapports et modèles de données Power BI en milieu utilisateur. Ces fonctionnalités sont requises par le libre-service et la conception affaires. Ce groupe AD inclura les différents groupes AD de super-utilisateurs et concepteurs affaires.

Options du service Power BI inclusDescription option
Utiliser des jeux de données dans plusieurs espaces de travailPermet de séparer les jeux de données des rapports.
Télécharger les rapportsPermet de télécharger une copie de rapports ou de jeux de données à partir du service Power BI.
Données par utilisateur dans les métriques d'utilisation des créateurs de contenuPermet de consulter les statistiques d'utilisation en voyant le nom des utilisateurs par espace de travail.
Autoriser les connexions activesPermet de se connecter à un jeu de données dans le service Power BI. Requis pour le libre-service, permet aussi l'analyse dans Excel ou libre-service Excel.

Groupe AD: GR-GIE-IA-PBI-LIBRE-SERVICE-EXCEL​

Ce groupe est créé pour gérer exclusivement l'accès à l'option de se connecter à un modèle de données publié dans le service Power BI à partir d'Excel. Une fois connecté à un modèle, l'utilisateur peut créer des tableaux croisés dynamiques sans avoir l'ensemble des données dans son fichier Excel.

Seul le résultat sera visible dans le fichier Excel. L'ensemble des données résident dans le jeu de données Power BI.

Cette option donne accès à l'ensemble du modèle de données et des mesures créées dans Power BI, mais via l'interface de tableau croisé dynamique d'Excel.

Cette option est recommandée pour les utilisateurs devant fréquemment avoir accès à des extractions pour exécuter des travaux ad hoc.

Excellente option pour rendre certains utilisateurs autonomes dans l'accès aux données et remplacer des extractions de données en fichier plat qui sont partagées avec les utilisateurs.

Options du service Power BI inclusDescription option
Autoriser les connexions activesAnalyse dans Excel ou libre-service-Excel : Permet d'utiliser les jeux de données Power BI comme source pour créer des tableaux croisés dynamiques dans Excel.

NOTE

Cette option étant nécessaire pour le développement en mode jeu de données partagé, elle est aussi incluse dans les groupes AD de gestion des options sécurisées suivants :

  • GR-GIE-IA-PBI-DEV-GR
  • GIE-IA-PBI-OPTIONS-DEV-AFFAIRES

Elle apparait dans la grille des options incluses sous le libellé :

  • Autoriser les connexions actives Analyser dans Excel

NOTE 2

En plus d'être membre d'un des groupes AD donnant accès à l'option autoriser les connexions actives, l'utilisateur (développeur, super-utilisateur, concepteur affaires, ou utilisateur du libre-service Excel) doit avoir accès au jeu de données publié dans le service Power BI avec le privilège Build

  • Voir pratique: Gérer la sécurité pour un jeu de données partagé

Groupe AD: GR-GIE-IA-PBI-EXPORT-EXCEL​

Ce groupe est créé pour gérer exclusivement l'accès à l'exportation vers Excel. Les membres de ce groupe auront accès à l'option exporter vers Excel dans les visuels de Power BI. Cette option permet d'extraire les données d'un visuel spécifique et de les afficher dans Excel.

Options du service Power BI inclusDescription option
Exporter vers ExcelExporte les données d'un visuel de rapport publié dans le service Power BI dans un fichier Excel sur le poste de l'utilisateur.

Groupe AD: GR-GIE-IA-PBI-EXPORT-PDF_PPT​

Ce groupe est créé pour gérer exclusivement l'accès à l'exportation de rapports publiés dans le service Power BI au format.pdf ou PowerPoint.

Les membres de ce groupe auront accès à l'option exporter vers Excel dans les visuels de Power BI. Cette option permet d'extraire les données d'un visuel spécifique et de les afficher dans Excel.

Options du service Power BIDescription option
Exporter vers ExcelExporte les données d'un visuel de rapport, publié dans le service Power BI, dans un fichier Excel sur le poste de l'utilisateur.

Groupe AD: GR-GIE-IA-PBI-SUPPORT​

Ce groupe est créé pour gérer exclusivement l'accès aux options du service Power BI dont l'équipe de soutien corporatif (le CSU) doit avoir accès.

Options du service Power BIDescription option
À venirCourriel d'erreur du Gateway. Permet d'informer le CSU quand le Gateway est en erreur afin que le CSU puisse informer ceux qui le contacteront de la raison pour laquelle les rapports ne présentent pas de données.
