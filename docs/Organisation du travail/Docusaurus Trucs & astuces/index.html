Noter que pour ce second jeu d'onglet, on a pas besoin de répéter le block de code ìmport Tabs

Info

Noter aussi que dans le document Markdown, les onglets ne sont pas visible et le texte html apparait. Les onglets sont visible seulement dans le site web.

><Tabs className="unique-tabs">
