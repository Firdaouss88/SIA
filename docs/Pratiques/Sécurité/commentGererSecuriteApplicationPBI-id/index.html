Comment gérer la sécurité d'une application Power BI

Gérer la sécurité d'une application Power BI​

Une fois que vous ouvrez votre application, vous pouvez la mettre à jour en ajouter les autorisations que vous voulez assigner à votre groupe ou aux utilisateurs.

Sur Autorisations, assigner le groupe AD de l'application sur l'application Power BI des rapports.

Vous pouvez autoriser à tous les utilisateurs qui ont accès à l'application à :​
  • Autorisez tous les utilisateurs à se connecter aux jeux de données sous-jacents de l'application à l'aide de l'autorisation de création.
  • Autorisez les utilisateurs à effectuer une copie des rapports dans cette application.
  • Autorisez les utilisateurs à partager l'application et les jeux de données sous-jacents de l'application à l'aide de l'autorisation de partage.

Quand vous sélectionnez Publier l’application , un message confirme que l’application est prête à être publiée. Sélectionnez Publier.

Dans la boîte de dialogue Publication effectuée, vous pouvez copier l’URL qui est un lien direct vers cette application.
