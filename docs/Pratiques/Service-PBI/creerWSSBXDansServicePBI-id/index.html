Créer un espace de travail d'exploration pour développeur: WS-SBX

Tâches​

  1. Créer l'espace de travail WS-SBX ;

  2. Publier le jeu de données ;

  3. Gérer la passerelle de données ;

  4. Configurer le partage du jeu de données ;

  5. Publier le rapport dans le WS-SBX ;

  6. Sélectionner les rapports à inclure dans l'application ;

  7. Créer l'application ;

  8. Gérer la sécurité.
